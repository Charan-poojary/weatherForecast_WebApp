## WeatherForecast React WebApp

### Published at https://weather-forecast-web-app.vercel.app/

## Software Features
* Current location Forecasting.
* Animated Video Backgrounds based on weather and time.
* Hourly Forecasting.
* Search locations.
* Air Conditions / Astronomy Details.
* Change accent/Theme.
